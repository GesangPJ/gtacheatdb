# GTA CHEATS DATABASE

A NextJS 14 Full Stack Web to provide cheats for some GTA Games.
Using MongoDB to save the data, plain JSON available.

## Game List

- GTA : III
- GTA : Vice City
- GTA : San Andreas
- GTA : IV
- GTA : V

## Platform Available (Depends On The Game)

- Desktop PC
- PlayStation
- XBOX
- Nintendo Switch
